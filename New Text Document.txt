How to Fix the Issue:
In Coolify, set these environment variables:
Go to your application settings in Coolify
Navigate to Environment Variables
Add:
TELEGRAM_BOT_TOKEN = your Telegram bot token
ADMIN_CHAT_ID = your Telegram chat ID
PORT = 3000 (optional, defaults to 3000)
After adding the variables, restart/redeploy your application
Check the application logs in Coolify — you should see:
✓ TELEGRAM_BOT_TOKEN is set
✓ ADMIN_CHAT_ID is set
If variables are missing, you'll see warnings in the logs.
